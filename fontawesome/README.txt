Coloque aqui os arquivos do Font Awesome baixados do site oficial:
https://fontawesome.com/download

1. Extraia o conteúdo do zip.
2. Copie a pasta 'webfonts' e o arquivo 'css/all.min.css' para dentro desta pasta 'fontawesome'.
3. No seu HTML, troque o link CDN por:
<link rel="stylesheet" href="fontawesome/css/all.min.css">

Assim, os ícones funcionarão mesmo offline.

Se quiser, posso ajustar o HTML automaticamente para você depois que os arquivos estiverem aqui.
